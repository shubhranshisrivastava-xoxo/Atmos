# Atmos
A modern weather web application that provides real-time weather updates for any city using the OpenWeather API.

## Features
- Search weather by city
- Current location weather
- Dynamic themes based on weather
- Recent searched cities
- City not found card
- Mobile responsive design
- Splash screen
- Add to Home Screen support

## Technologies Used
- HTML
- CSS
- JavaScript
- OpenWeather API
